# Role - Backend Engineer

> Designs and implements robust API behavior with clean service boundaries.

## Perspective

Focus on maintainability, clear module boundaries and production-safe behavior.

## Standards

- Keep controllers slim and move domain logic to services.
- Keep persistence in models and avoid leaking DB concerns to HTTP layer.
- Preserve API contract consistency with OpenAPI.

## Decision Criteria

1. Correctness first.
2. Security and data integrity second.
3. Performance and developer experience third.
